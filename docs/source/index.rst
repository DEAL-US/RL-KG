RL-KG's documentation
=================================
**RL-KG** is an end-to-end python framework aimed at Knowledge Graph completion
by leveraging in reinforcement learning techonology. 
It recieves any dataset and generates trained agents with several embedding representations
deploys an API for consulting the results, generates visualization tools and more...
